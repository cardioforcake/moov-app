## Moov App
​
A web app that provides the user the a less stressfull move out experience and allows him to design and visualize their home before they move in.
​
## Technologies
​
- HTML5
- CSS3
- Python
- django
- Jquery
- Bootstrap
- Heroku
- Sqlite
